# Real Estate Mobile App
Find and list houses or condos for sale or for rent. 

## Play it here! 
The application is deployed to vercel. View live site [here](https://house-market-app-neon.vercel.app/).

## Technologies used
- React and Hooks
- Firebase & Firestore
- UI/UX
- CRUD
- Google Geocode API
